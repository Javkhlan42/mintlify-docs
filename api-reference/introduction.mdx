## Танилцуулга

Yellow Book API нь байгууллага, хэрэглэгч болон гуравдагч системүүдийн хооронд
өгөгдөл солилцох зориулалттай RESTful API юм.

Энэхүү API нь байгууллагын лавлах мэдээлэл, хайлт, AI Assistant-д суурилсан
асуулт-хариултын үйлчилгээг системүүдэд интеграци хийх боломж олгоно.

---

## Ерөнхий мэдээлэл

Yellow Book API нь дараах зорилготой:

- Байгууллагын мэдээллийг төвлөрсөн хэлбэрээр дамжуулах  
- Directory болон ангилалд суурилсан хайлт хийх  
- AI Assistant ашиглан асуултад суурилсан лавлагаа авах  
- Веб, mobile болон backend системүүдтэй холбогдох  

API нь **JSON форматтай** өгөгдөл солилцоно.
