## Error Handling

<AccordionGroup>
  <Accordion title="404 Not Found" icon="circle-xmark">
    **Шалтгаанууд:**
    - Буруу ID оруулсан
    - Байгууллага устгагдсан эсвэл идэвхгүй болсон
    
    **Шийдэл:**
    - ID зөв эсэхийг шалгах
    - `/directory` endpoint-аар бүх байгууллагын жагсаалт авах
  </Accordion>
  
  <Accordion title="400 Bad Request" icon="triangle-exclamation">
    **Шалтгаанууд:**
    - ID формат буруу
    
    **Шийдэл:**
    - ID string төрөлтэй эсэхийг шалгах
    - Хоосон ID илгээгээгүй эсэхийг шалгах
  </Accordion>
  
  <Accordion title="429 Too Many Requests" icon="clock">
    **Шалтгаанууд:**
    - Rate limit хэтэрсэн
    
    **Шийдэл:**
    - `retryAfter` секунд хүлээх
    - Хүсэлтийн тоог багасгах
  </Accordion>

## Code Examples

<CodeGroup>

```javascript JavaScript - Basic
async function getOrganizationDetails(organizationId) {
  const response = await fetch(
    `https://sharnom.systems/organization/${organizationId}`
  );
  
  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }
  
  const data = await response.json();
  
  if (data.status === 'Success') {
    return data.value;
  } else {
    throw new Error(data.msgList.join(', '));
  }
}

// Ашиглалт
try {
  const org = await getOrganizationDetails('1');
  console.log('Байгууллага:', org.name);
  console.log('Утас:', org.phone);
} catch (error) {
  console.error('Алдаа:', error.message);
}
```

```python Python - requests
import requests

def get_organization_details(org_id):
    url = f"https://sharnom.systems/organization/{org_id}"
    response = requests.get(url)
    
    if response.status_code == 200:
        data = response.json()
        if data['status'] == 'Success':
            return data['value']
        else:
            raise Exception(', '.join(data['msgList']))
    else:
        response.raise_for_status()

# Ашиглалт
try:
    org = get_organization_details('1')
    print(f"Байгууллага: {org['name']}")
    print(f"Утас: {org['phone']}")
except Exception as e:
    print(f"Алдаа: {e}")
```

```typescript TypeScript
interface Organization {
  id: string;
  name: string;
  category: string;
  address: string;
  district: string;
  phone: string;
  email?: string;
  website?: string;
  latitude?: number;
  longitude?: number;
  description?: string;
  workingHours?: string;
}

interface OrganizationResponse {
  value: Organization;
  data: null;
  msgList: string[];
  attrs: Record<string, any>;
  status: 'Success' | 'Error';
}

async function getOrganizationDetails(
  organizationId: string
): Promise<Organization> {
  const response = await fetch(
    `https://sharnom.systems/organization/${organizationId}`
  );
  
  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }
  
  const data: OrganizationResponse = await response.json();
  
  if (data.status === 'Success') {
    return data.value;
  } else {
    throw new Error(data.msgList.join(', '));
  }
}
```
